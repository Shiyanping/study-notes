# jQuery 的链式开发

var example = {
  a: function() {
    console.log('first');
    return this;
  },
  b: function() {
    console.log('second');
    return this;
  },
  c: function() {
    console.log('third');
  }
};
example
  .a()
  .b()
  .c();

在每个方法执行完成后返回当前大的对象，使得下一个方法调用时，this 的指向还是当前大对象。
