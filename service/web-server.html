# Web 服务器基础原理

用户访问指定的域名，域名通过 DNS 解析出来 IP，对应的服务器下通过负载均衡及内容分发，将用户分配到最近压力最小，访问最快的服务器上，服务器下发访问的文件，然后返回给用户。

关键词：

  • 负载均衡
  • 内容分发
