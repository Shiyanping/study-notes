# Linux 常用命令

- `ls` 或者 `dir`

  查看当前文件夹下面有哪些文件（不包括隐藏文件）

- `ls -l`

  查看文件夹下面文件的详细内容（不包括隐藏文件）

- `ls -a`

  查看文件下面所有的内容（包括隐藏文件）

- `mkdir 文件夹名`

  创建一个目录（不同创建同名的文件夹）

- `touch 文件名`

  创建一个文件

- `cd`

  进入某个文件夹，退出一个文件夹，在`cd`后面添加`..`

- `cp 需要复制哪个文件 复制到哪`

  复制一个文件（可以写目录），想要复制一个文件夹，只需要在`cp`后面添加`-R`

- `pwd`

  输出当前文件目录地址

- `rm 文件名`

  删除一个文件（不会有任何提示），想要删除一个文件夹在`rm`后面添加`-r`
