# 常用 npm 命令

  • npm -v：查看 npm 版本号
  • npm list -g --depth 0：查看全局安装的 npm 包
  • npm uninstall XX：删除一个 npm 包，添加-g 可删除全局包
  • npm search XX：查询 npm 包
  • npm help：查看 npm 帮助，可在后面添加指定命令的帮助，如：npm help install
